## Question

### **n까지 거리 겹치지 않고 이동하기 (실패)**

## Solve

```java
class Solution1 {
  public int solution(int n, int k) {
    double cnt = 0;

    // 갈 수 있는 거리
    int[] l = new int[k];
    for (int i = 0; i < l.length; i++) {
      l[i] = k - i;
    }

    for (int i = 0; i < l.length; i++) {
      int sum = l[i]; // 움직인 거리
      int pre = l[i]; // 이전에 움직인 거리

      while (sum < n) {
        for (int j = 0; j < l.length; j++) {
          int x = sum + l[j]; // 선택한 거리만큼 움직였을 때 도달한 거리
          if (l[j] != pre && x <= n) {
            sum += l[j];
            pre = l[i];
          }

          if (sum == n) {
            cnt += (cnt + 1) % 1000000007;
          } else if (sum > n) {
            break;
          }
        }
      }
    }
    return (int) cnt;
  }
}
```

## Question

### **가장 큰 연속합**

## Solve

```java
class Solution2 {
  public int solution(int[] A) {
    // 음수의 경우 0으로 반환한다.
    int max = -1;
    int sum = 0;

    for (int i = 0; i < A.length; i++) {
      sum = A[i];
      max = Math.max(max, sum);
      for (int j = i+1; j < A.length; j++) {
        sum += A[j];
        max = Math.max(max, sum);
      }
    }
    return (max >= 0) ? max : 0;
  }
}
```

## Note

```java
class Solution3 {
  public int[][] solution(int[][] n) {
    // 버스정류장 개수 세기
    int cnt = 0;
    for (int[] a : n) {
      for (int x : a) {
        if (x == 0) {
          cnt++;
        }
      }
    }

    // 버스정류장 인덱스 배열 따로 저장
    int[][] bus = new int[cnt][2];
    cnt = 0;
    for (int i = 0; i < n.length; i++) {
      for (int j = 0; j < n[i].length; j++) {
        if (n[i][j] == 0) {
          bus[cnt][0] = i;
          bus[cnt++][1] = j;
        }
      }
    }

    // 정류장 까지의 거리 저장 (기존 배열n에 덮어쓰기)
    for (int i = 0; i < n.length; i++) {
      for (int j = 0; j < n[i].length; j++) {
        if (n[i][j] != 0) {
          n[i][j] = getShortLength(bus, i, j);
        }
      }
    }
    return n;
  }

  // 버스정류장 인덱스만 따로 저장한 배열에서 하나씩 꺼내며 가장 가까운 거리 구하기
  public int getShortLength(int[][] bus, int i, int j) {
    int min = Integer.MAX_VALUE;
    for (int a = 0; a < bus.length; a++) {
      min = Math.min(Math.abs(bus[a][0]-i) + Math.abs(bus[a][1]-j), min);
    }
    return min;
  }
}
```

## Question

### **1이 연속으로 나타나는 최대 길이**

1011 : 2

## Solve

```java
class Solution4 {
  public int solution(int[] arr) {
    // 바로 직전 검사했던 길이
    int pre = 0;
    // 현재 검사하는 길이
    int sum = 0;

    for (int i = 0; i < arr.length; i++) {
      // 1이 나오면 카운팅
      if (arr[i] == 1) {
        sum++;
      // 0이 나오면 지금까지 카운팅했던 길이를 pre 에 저장
      // sum은 다시 0으로 초기화
      } else {
        pre = Math.max(pre, sum);
        sum = 0;
      }
    }
    // [0110111] 의 경우 pre에 저장하지 못하고 반복문을 나오기 때문에 둘 중에 큰 값을 반환
    return Math.max(pre, sum);
  }
}
```

## Question

### **돌다리를 건널 수 있는지 여부**

최대 길이 : 2, 돌다리 : 1, 없음 : 0

110001 : 실패

## Solve

```java
class Solution {
  public boolean solution(int[] bridge, int jumpSize) {
    // 돌이 없는 구간의 길이
    int cnt = 0;

    for (int i = 0; i < bridge.length; i++) {
      // 돌이 없는 구간 카운팅
      if (bridge[i] == 0) {
        cnt++;
      // 다시 돌다리가 나오면 0으로 초기화
      } else {
        cnt = 0;
      }
      // 돌이 없는 구간이 최대길이를 넘으면 바로 false 반환
      if (cnt > jumpSize) {
        return false;
      }
    }

    // 끝까지 걸러지지 않고 반복문을 탈출하면 true 반환
    return true;
  }
}
```
